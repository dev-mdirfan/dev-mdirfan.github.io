# Escape Sequences:

- `\\`   -->   To print Backslash \
- `\"`   -->   Double Quote " 
- `\'`   -->   Single Quote '
- `\n`   -->   New Line
- `\t`   -->   Tab
- `\r`   -->   Carriage Return
- `\b`   -->   Backspace
- `\f`   -->   From Feed

```python
print("hi\\This\"is\'machine\nlearning\tusing\bpython\fin\nenglish\rhindi")
```
